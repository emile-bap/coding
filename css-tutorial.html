CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. With CSS, you can change colors, fonts, layouts, and even create animations for HTML elements.

CSS Syntax

CSS is written in the format: selector { property: value; }

                
h1 {
    color: blue;
    text-align: center;
}

In this example, we are selecting the h1 tag and changing its color and text-align.

Selecting Elements

You can style different elements in HTML by using selectors:

  • Type Selector: Targets all elements of a particular type (e.g., p selects all paragraphs).
  • Class Selector: Targets elements with a specific class (e.g., .classname selects all elements with the class "classname").
  • ID Selector: Targets a single element with a specific ID (e.g., #idname selects the element with the ID "idname").

CSS Box Model

The CSS Box Model describes how each element on a web page is structured. It consists of:

  • Content: The actual content of the box, where text and images appear.
  • Padding: The space between the content and the border.
  • Border: A border surrounding the padding and content.
  • Margin: The space outside the border, separating the element from other elements.
                
.box {
    width: 300px; 
    padding: 20px; 
    border: 5px solid black; 
    margin: 15px;  
}

Responsive Design with Media Queries

Media queries allow you to apply CSS styles based on the screen size or device type. For example:

                
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

This rule changes the background color of the body to light blue when the screen is 600 pixels or smaller.
